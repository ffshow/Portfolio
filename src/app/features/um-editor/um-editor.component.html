User Mapping Editor - Refactoring (App JAVA)

Pour répondre à une problématique de maintenabilité d'un outil déjà existant, il m'a été demandé d'en faire une analyse et voir s'il était possible d'en améliorer l'affichage et d'ajouter des fonctionnalités supplémentaires notamment la prise en charge du catalogue de script JS. Afin de répondre à cette problématique j'ai décidé d'en effectuer le refactoring*.

Tout d'abord, à quoi servait l'outil ?
Cette application (UserMappingEditor) est un outil qui permet la simulation d’envoi de message avec des paramètres d’entré et de sorti (Format du message, format d’encodage, l’application de règle CSV ..).
Afin de rendre l’application beaucoup plus lisible au niveau du code et de rendre sa maintenance optimale (ajout de fonctionnalités) il m’a été demandé d’en effectuer le refactoring.

Version Java : Java SE 8 (JDK 1.8.0_171)

*Réusinage de code est l'opération consistant à retravailler le code source d'un programme informatique.
