Développement WEB

Description

Le développement web repose sur l’utilisation des langages (HTML/CSS, JavaScript, PHP…) pour écrire des programmes qui sont ensuite exécutés par les ordinateurs. Les instructions sont mises en place sur Internet et sont effectuées sur des serveurs. En fonction des besoins des propriétaires du site ou des pages web, ces dernières peuvent être constituées uniquement de textes et d’éléments graphiques ressemblant à un document ou être interactives en affichant des informations qui évoluent (panier d’achat, par exemple).

Mise en contexte

Durant mes années d'alternance avec la montée en compétence en javascript notament grace à la constitution du catalogue de script, j'ai effectué une formation en sur un framework JS Angular associé avec Bootstrap etc.. pour la réalisation de ce portfolio, en intervenant aussi sur le front du logiciel BCI Connect par exemple mais aussi pour la création de mon projet perso Pulpe!.

Mon point de vue

J’aime le processus créatif qu’il est possible d’avoir dans le développement web (front), la satisfaction de voir son code se transformer au beau ramassis de pixel bien organisé et fonctionnels.
